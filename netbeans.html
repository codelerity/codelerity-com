Apache NetBeans packages / installers provided by Codelerity and Gj IT (see who we are). These packages are built with NBPackage using the official Apache NetBeans binary zip release. The packages themselves are not released by the Apache Software Foundation.

These installers include a local Java 21 JDK for the IDE to run on, offering a self-contained out-of-the-box experience. Additional JDK's for projects may be registered and/or downloaded using Tools / Java Platforms in the IDE menu, supporting development of projects for Java 8 up to Java 21.

These packages are provided without warranty, and under the licenses and terms of the bundled software. Please make sure you're familiar with all terms before downloading. Apache NetBeans is provided under the terms of the Apache Software License. Some of the packages include a build of OpenJDK - Azul Zulu for x86_64, and BellSoft Liberica for Arm - please see terms and licenses linked by the relevant downloads.

Support

For issues with the installers / packages themselves, please raise an issue on our GitHub repository.

For community support with the IDE itself, please use the Apache NetBeans mailing lists.

For commercial support, maintenance or development with the IDE or platform, please contact us.
We can also provide customised and supported installers with a JDK and cluster configuration of your choice.

Your sponsorship helps provide these installers, and your support is gratefully received!

Linux (Arm)

We are currently reevaluating our support for the Raspberry Pi and similar Arm machines. In the meantime we recommend using the upstream all architecture packages with a system installed JDK.

Linux (any architecture)

Now available directly from Apache NetBeans.

Following our contribution of NBPackage to the Apache NetBeans project, RPM and DEB packages requiring and using a system installed JDK will be made available directly from Apache NetBeans in future.
