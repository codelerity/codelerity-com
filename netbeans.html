Apache NetBeans packages / installers provided by Codelerity and Gj IT (see who we are). These packages are built with NBPackage using the official Apache NetBeans binary zip release. The packages themselves are not released by the Apache Software Foundation.

These installers include a local Java 17 (LTS) JDK for the IDE to run on, offering a self-contained out-of-the-box experience. Additional JDK's for projects may be registered and/or downloaded using Tools / Java Platforms in the IDE menu, supporting development of projects for Java 8 up to Java 19.

These packages are provided without warranty, and under the licenses and terms of the bundled software. Please make sure you're familiar with all terms before downloading. Apache NetBeans is provided under the terms of the Apache Software License. Some of the packages include a build of OpenJDK - Azul Zulu for x86_64, and BellSoft Liberica for Arm - please see terms and licenses linked by the relevant downloads.

Linux (x86_64)

These packages include Azul Zulu build of OpenJDK - license terms.

Apache NetBeans 16 + JDK 17 (.deb x86_64)
The .deb package should be installed using the distribution software manager, or using eg. sudo dpkg -i FILENAME.
Apache NetBeans 16 + JDK 17 (.rpm x86_64)
The .rpm package should be installed using the distribution software manager, or using eg. sudo dnf install ./FILENAME.
Apache NetBeans 16 + JDK 17 (.AppImage x86_64)
The AppImage does not require installation or root access - simply make executable and run.
